Running in Eclipse 
-----------------
1. Unzip CityRoadMap.zip file and Export Maven Project to Eclipse
2. Run as com.bank.master.MasterCardApplication.java as Spring boot App or Java Application 
3. Test URL http://localhost:8080/connected?origin=Boston&destination=Newark
4. Run as JUnit Test CityConnectRestApplicationTest.java for Junit without starting Server

Maven Insatll
--------------
1. Unzip CityRoadMap.zip file and Export Maven Project to Eclipse
2. Right click MasterCard Project -->Run As --> Maven Install 
    Please find java jar under target/MasterCard-0.0.1-SNAPSHOT.jar
